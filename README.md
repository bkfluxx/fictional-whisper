# Fictional Whisper

A self-hosted, privacy-first personal journal and notes application.

- **Stack**: Next.js (App Router) · TypeScript · PostgreSQL · Prisma · Tailwind CSS
- **AI**: Ollama (fully local — no cloud APIs)
- **Security**: AES-256-GCM encryption at rest · NextAuth single-user auth
- **Deployment**: Docker Compose

See [PRD.md](./PRD.md) for full product requirements.
